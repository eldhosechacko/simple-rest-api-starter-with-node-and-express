# Simple REST API with Node & Express

A minimal, modular REST API demo built with Node.js and Express.

Features:
- Modular structure: `src/app.js`, `src/routes`, `src/controllers`
- Base route (`GET /`) returns a JSON welcome message

Quick start

Install dependencies:

```bash
cd "/Users/eldose/MyWorks/for YouTube/simple rest api with node and express"
npm install
```

Run:

```bash
npm start
# or for development with automatic reload:
npm run dev
```

Then open http://localhost:3000/ to see the JSON welcome message.
